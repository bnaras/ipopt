# Ipopt package library from COIN OR tools which can solve nonlinear optimization problems

Provides a lightweight R interface to the Ipopt nonlinear optimization
solver through Ipopt's C interface. The package is intended for
r-universe builds that provide Ipopt and its open linear solver
dependencies, such as MUMPS, through system libraries discoverable by
pkg-config.
