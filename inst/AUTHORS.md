# Authorship and Credits

This R package provides a lightweight interface to the Ipopt library. The R
wrapper code is maintained by Balasubramanian Narasimhan.

Ipopt itself is a COIN-OR project. The Ipopt distribution identifies Andreas
Waechter and Carl Laird as the main authors. The Ipopt README identifies
Andreas Waechter and Stefan Vigerske as COIN-OR project managers for Ipopt.

For the complete upstream Ipopt contributor list, see the `AUTHORS` file in the
Ipopt source distribution and the project repository:
<https://github.com/coin-or/Ipopt>.
